# Notebooks

Exploratory Jupyter notebooks. Filled in during Phases C and E.

- `01_explore_distributions.ipynb` — (Phase C) histograms of per-rep angle,
  ROM, and tempo stats. Sanity check before percentile derivation.
- `02_validate_thresholds.ipynb` — (Phase E) before/after plots comparing
  proposed derived thresholds against the current literature defaults,
  using the Dart replay output.

Commit notebooks with output cells cleared (`jupyter nbconvert --clear-output`)
to keep diffs reviewable.
